Stock Market Investments Sheet
Click for demo video

A personal tool based on retrieval of stock prices and financial indicators from different API's, calculation of technical analysis indicators in R with functions written on the resources from "quantmod" and "dplyr" packages. These help me score the financial and technical analysis of stocks to take buy, hold, and sell positions. I can automate the calculation of the return on investment of overall, active, or closed stock investments. This part was a bit trickier to calculate, since I can hold active stocks of the same asset which I previously closed a transaction of. But it turns out that DAX (language of PowerBI tables) is more capable than it seems to be.

Socioeconomy and Consumer Trends of Nearby Countries
Click for demo video

A PowerBI work consists of the data queried from "Worldbank" & "Oxford Economics" API's, transformed into a uniform table. It is a decision making support tool about countries near Turkey. For instance, user can not only say GDP is increasing, but also claim if lower income groups directly benefit from GDP growth. To draw these conclusions, I had to figure out how to provide the user the ability to choose a single indicator or the ratio of two different indicators. Eventually I ungrouped all the data in seperate tables and let user calculate the ratio of the selected indicators.

VR Property Touring Application
Click for demo video

A Node.js web application which uses two different open source VR libraries: "A-frame" and "Panellum". I chose "jquery" to empower the application with interactive user operations. The application offers virtual tour of properties with dynamic location pins on the building layout. It is both browser and VR headset compatible. I have performed the entire development pipeline of the project: Requirements research, UX design, photoshooting, photo editing, application development, user acceptance.

Shopping Center Management Weekly KPI Analysis Dashboard

A dynamic PowerBI solution consists of calculation of weekly KPI's in areas such as leasing, visitors, revenue & collections for each shopping center in the management portfolio. Microsoft Navision ERP datasets has been used as a source.

Shopping C. Tenants Map

A project consists of collection of 23,000+ shopping center tenant information across Turkey, cleansing data and then visualization with PowerBI. This work provides actionable insights such as geographic penetrations of retail brands and competition analysis of the same category brands.

Dynamic P&L Statement

A dynamic PowerBI model that provides insights on the KPI's of productivity, renevue, ageing, collections and the overall financial state of a consultancy company in Turkey. ERP SQL server data has been used as the source data.

Shopping C. Feasibility Model

A PowerBI work which shows the current state analysis of shopping center leasing feasibility which calculates varying KPI's about vacany, brand mix, rent amount projection. Dynamics CRM data has been used as a source.

Office Floor Plan SSRS Report

A Dynamics CRM SSRS report designed to summarize the layout of an office building in terms of vacancy, tenants and the rental figures. The CRM data used as a source contains property, floor availabilities, and lease records. It was a tough challenge to calculate vacancies of the floors with indirect relationship to lease data within the limitations of SSRS.
